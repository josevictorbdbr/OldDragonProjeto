# CRIAÇÃO DE UM PERSONAGEM BASEANDO-SE NAS REGRAS DO RPG OLD DRAGON

(Recomendado: Para iniciar o codigo digite "cmd" e "python Main.py" para funcionar caso o arquivo .venv não funcione)

Este projeto é um sistema em Flask para criação e armazenamento de personagens inspirados no RPG Old Dragon.  
O usuário pode escolher raça, classe e estilo de distribuição de atributos, gerar a ficha e salvá-la como JSON.
