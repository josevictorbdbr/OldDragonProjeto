# Criador de ficha de personagem para RPG

Este projeto é um sistema em Flask para criação e armazenamento de personagens inspirados no RPG Old Dragon.
O usuário pode escolher raça, classe e estilo de distribuição de atributos, gerar a ficha e salvá-la como JSON.

Old Dragon: https://olddragon.com.br

## Como rodar: 
- Clone o repositorio
- Instale o ambiente virtual(venv) com `Flask` que esta no arquivo `requirements.txt`
- Rode o arquivo `main.py` e abra o link que aparecer no terminal
- Crie seu personagem
